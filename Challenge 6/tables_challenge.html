Specific Uses of Tabular Data

  • Publishing Table

    • Publishing tables are mirror tables that you have identified for monitoring. They contain additional columns,
    primarily a sequence number and delivery status, which are required by the adapter to detect new rows.

  • Mathematics Table

    • Mathematical tables are lists of numbers showing the results of calculation with varying arguments.

  • Periodic Table

    • The Periodic Table is a table of the chemical elements arranged in order of atomic number, usually in
    rows, so that elements with similar atomic structure (and hence similar chemical properties) appear
    in vertical columns.

  • Databases

    • A database is an organized collection of data, stored and accessed electronically.

  • Software Applications

    • In computer programming, a table is a data structure used to organize information, just as it is on paper.
